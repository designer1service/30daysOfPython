## Math operators I practiced

I used Python arithmetic operators in small examples:

- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division (always returns a decimal / float result)
- `%` modulo (returns the remainder of a division)
- `//` floor division (division rounded down to a whole number)
- `**` exponent (power)

### Operator notes (quick mental model)

- `%` gives the remainder after division (useful for checking if a number is even/odd).
- `//` performs floor division, meaning it drops the fractional part and returns only the integer.
- `**` raises a number to a power (e.g., `2 ** 3` is 8).
