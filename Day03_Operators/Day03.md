## Key Concepts Covered

### 1. Arithmetic Operators
- `+`, `-`, `*`, `/` (standard math, `/` always returns a float)
- `//` (floor division, drops the decimal)
- `%` (modulus, gives the remainder)
- `**` (exponentiation / power)

### 2. Assignment & Bitwise Operators
- Standard assignment (`=`, `+=`, `-=`, etc.)
- Basic bitwise operators like `&=` (AND), `|=` (OR), `^=` (XOR), and `>>=` (Right Shift).

### 3. Comparison Operators
I learned how to compare values, which always results in a Boolean (`True` or `False`):
- `==` (equal), `!=` (not equal)
- `>`, `<`, `>=`, `<=`
- **Identity & Membership Operators:**
  - `is` / `is not` (checks if objects are identical)
  - `in` / `not in` (checks if a sequence or substring exists inside another, e.g., `'A' in 'Ana'`)

### 4. Logical Operators
- `and` (returns True if both statements are true)
- `or` (returns True if one of the statements is true)
- `not` (reverses the result)
